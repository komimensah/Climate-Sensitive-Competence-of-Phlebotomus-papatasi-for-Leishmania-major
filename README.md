# Climate-driven ZCL risk modelling

This repository contains the R code used to generate all analyses and
figures in the manuscript:

"Climate-driven transmission risk of zoonotic cutaneous leishmaniasis
mediated by Phlebotomus papatasi in North Africa"

## Contents
- scripts/   R scripts for analysis and figures
- figures/   Output figures
- data/      Input data (or instructions)

## Reproducibility
All figures can be reproduced by running the scripts in numerical order.
